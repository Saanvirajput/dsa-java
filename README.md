# dsa-java

A repository for practising Data Structures and Algorithms using Java.

## Structure

- `src/main/java/dsa/`: Contains all problem solutions categorized by topic and difficulty.
- `notes/`: Conceptual notes and cheat sheets.
- `progress.md`: Tracks problem-solving progress.

## Guidelines

1. **File Naming**: Use `PascalCase` (e.g., `TwoSum.java`).
2. **Packages**: Each topic is its own package (e.g., `package dsa.arrays.easy;`).
3. **Difficulty**: Problems should be placed in `easy`, `medium`, or `hard` folders.
